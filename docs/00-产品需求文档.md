# 梦见小程序 PRD v1.0

## 一、产品概述

### 1.1 产品定位

一款结合 AI 梦境解析与 gamification 机制的每日打卡类小程序，通过"记梦-解析-改运任务"的闭环，培养用户习惯并通过激励视频广告变现。

### 1.2 目标用户

- 核心用户：18-35 岁女性，对玄学/心理学有兴趣
- 次要用户：有解梦需求的泛用户群体

### 1.3 商业模式

- 激励视频广告（查看完整解析前必看）
- 小额会员（9.9 元/月解锁无广告+高级功能）
- Banner 广告（非核心流程）

### 1.4 产品介绍

每个梦都是心灵的私语。在这里，记录夜晚的奇遇，AI 为你温柔解读梦境密码。坚持记梦，遇见更懂自己的你。

## 二、核心功能

### 2.1 MVP 功能清单

#### 功能 1：梦境记录

**入口**：首页"记录今天的梦"按钮

**交互流程**：

1. 用户输入梦境描述（文字，50-1000 字）
2. 可选：添加梦境标签（追逐/飞行/考试/亲人等 10 个预设标签）
3. 可选：选择情绪（开心/恐惧/困惑/悲伤）
4. 点击"开始解析"

**数据存储**：

- 梦境文本
- 标签
- 情绪
- 记录时间

#### 功能 2：AI 梦境解析

**触发条件**：用户提交梦境后

**交互流程**：

1. 显示"AI 解析中..."加载动画（3 秒）
2. 弹出激励视频广告（30 秒）
3. 观看完毕后展示解析结果

**解析内容结构**：

- **梦境主题**：1 句话概括（如"对未知的焦虑投射"）
- **心理解读**：3-5 句话的专业分析
- **今日运势**：运势评分（1-100 分）+ 3 个维度小 tips（事业/感情/健康）
- **改运任务**：1 个趣味任务（如"今天戴红色饰品"/"喝咖啡加糖"）

**技术方案**：

- 调用大模型 API（如通义千问/文心一言）
- Prompt 设计包含：周公解梦知识库 + 心理学常识 + 运势生成规则

#### 功能 3：梦境日历

**入口**：底部导航"我的"

**展示内容**：

- 日历视图，标记已记梦的日期
- 点击日期查看历史梦境
- 显示连续记梦天数徽章

#### 功能 4：改运任务打卡

**交互流程**：

1. 解析结果页显示今日任务
2. 用户完成后点击"已完成"
3. 弹窗奖励"幸运值+10"（虚拟积分）
4. 积分可用于：
    - 解锁历史梦境再解析
    - 兑换特殊解析（如"梦见 ta 是什么意思"专题）

## 四、广告策略

### 4.1 广告位设置

| 位置       | 类型     | 时机             | 预估 eCPM |
| ---------- | -------- | ---------------- | --------- |
| 解析前     | 激励视频 | 必看             | ¥80-150   |
| 首页底部   | Banner   | 常驻             | ¥5-10     |
| 任务完成后 | 激励视频 | 可选（双倍奖励） | ¥80-150   |

### 4.2 用户激励设计

- **必看广告**：查看解析结果前
- **可选广告**：观看后可获得双倍幸运值/额外解析次数

## 六、技术方案

### 6.1 技术栈

- 前端：uni-app (Vue 3 + Vite + Pinia)
- 后端：NestJS 10 + Prisma 5
- 数据库：MySQL 8 + Redis 7
- AI 能力：调用第三方大模型 API

### 6.2 成本估算（月活 1 万用户）

- 服务器：¥200/月
- AI API 调用：¥500/月（按 0.05 元/次，日均使用 1 次计算）
- 广告 SDK 接入：免费
- **总成本**：¥700/月

### 6.3 收入预估（月活 1 万用户）

- 广告收入：1 万人 × 30 天 × 2 次 × eCPM¥100/1000 = ¥60,000
- 会员收入：1 万人 × 2%转化 × ¥9.9 = ¥1,980
- **总收入**：¥61,980/月

**预估净利润**：¥61,280/月

## 八、风险与应对

### 8.1 核心风险

| 风险          | 影响         | 应对方案              |
| ------------- | ------------ | --------------------- |
| AI 解析质量差 | 用户流失     | 人工审核+优化 Prompt  |
| 广告完成率低  | 收入不达标   | 调整激励机制          |
| 留存率低      | 无法形成习惯 | 增加推送提醒+签到奖励 |

### 8.2 合规风险

- 避免使用"算命""预测"等敏感词
- 解析内容添加免责声明："本解析仅供娱乐参考"
- 不涉及付费占卜等需要资质的服务
